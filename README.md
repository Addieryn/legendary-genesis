## Dice 

### Rules 
- dice as modifiers
- 3 d4 modifiers with plus, minus, crit hit, crit miss
    - majority wins:
```
    1. + + - = +
    2. + + C = +
    3. + + + = +
    4. - - - = -
    5. - - + = -
    6. - - C = -
    7. + - C = 0
    8. C C -/+ = C
    9. C C C = C+/-
```
